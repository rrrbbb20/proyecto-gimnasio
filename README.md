# proyecto-gimnasio
Proyecto Fullstack 2026
Este proyecto consiste en una en una Arquitectura de Microservicios 
diseñada para gestionar las operaciones de un gimnasio. 
permite la administracion de clientes, planes, entrenamientos, inventario de equipos mediante la comunicacion entre 
servicios usando WebClient.

Requisitos de Ejecución
Para levantar el entorno, se requiere:
Java 17+ y Maven. 

Base de Datos: MySQL configurado segun el application.properties de cada servicio.  
IDE: IntelliJ o VS Code con extensiones de Spring Boot.

Pasos para ejecutar:
Clonar el repositorio.

Ejecutar cada microservicio de forma independiente comenzando por ms-auth.
Realizar pruebas de endpoints mediante Postman.
